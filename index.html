ABOUT ME

Hi, I am Natalie

Future Front End Developer based in Vancouver

My journey into coding started just about one year ago from Udemy courses, but it is not easy to learn code by yourself that's why I decided to enroll to correlation one program. If you want to become a web developer the first step to do that is learning how to become a good front-end developer, and that means, you have to learn first HTML, CSS, and JS. In my opinion for now Junior Front End developer should be familiar width:

  • HTML & CSS
  • CSS Preprocessors (Less & Sass)
  • JavaScript
  • Responsive Design
  • Git
  • React
  • Redux
  • NodeJS

One last thing, learning is not enough, you have to put in practice the concepts, how? well, for instance, cloning a web page you like and see how much you apply what you have learned. If you do not practice you will forget what you learned.
